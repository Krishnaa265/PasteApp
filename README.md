# Paste App

Paste App is a simple and efficient paste management web application built using React, Redux Toolkit, and React Router.
It allows users to create, edit, view, search, copy, and delete text pastes in a clean and user-friendly interface.

# Features

Create new pastes with a title and content

Edit existing pastes using URL query parameters

View a single paste on a dedicated route

Search pastes by title

Copy paste content to clipboard

Delete pastes

Automatically stores paste creation date

Client-side routing with React Router

Centralized state management using Redux Toolkit

Toast notifications for user actions

# Tech Stack

Frontend: React (Vite)

Routing: React Router v6

State Management: Redux Toolkit

Styling: Tailwind CSS

Icons: Lucide React

Notifications: react-hot-toast


# Installation and Setup

Clone the repository

git clone https://github.com/your-username/paste-app.git


Navigate to the project directory

cd paste-app


Install dependencies

npm install


Start the development server

npm run dev


The application will run on:

http://localhost:5173
